# 🍋 Co-Pay Calculator · คำนวณร่วมจ่าย

เครื่องคำนวณสำหรับ**โครงการร่วมจ่าย** (Co-Payment Scheme) ของรัฐบาล ช่วยให้ลูกค้าและร้านค้าคำนวณยอดเงินได้อย่างรวดเร็วและถูกต้อง

---

## 📋 ภาพรวมโครงการ

โครงการร่วมจ่ายแบ่งการชำระเงินระหว่างรัฐกับประชาชนในสัดส่วน:

| ผู้จ่าย | สัดส่วน |
|--------|--------|
| รัฐบาล | 60% |
| ประชาชน | 40% |

> วงเงินสูงสุดต่อวัน **333 บาท** (บางร้านค้า 334 บาท)

---

## ✨ ฟีเจอร์

- **แท็บลูกค้า** — คำนวณวงเงินซื้อสินค้าสูงสุด และยอดที่ต้องจ่ายเองจากราคาสินค้า
- **แท็บร้านค้า** — คำนวณเงินสดที่ต้องทอนให้ลูกค้า หลังหักค่าสินค้าและค่าธรรมเนียม
- **วิธีคิดแบบ Dynamic** — กดปุ่ม "วิธีคิด" เพื่อดูสูตรพร้อมตัวเลขจริงที่กรอกไว้
- **Responsive** — ใช้งานได้ดีทั้งมือถือและเดสก์ท็อป

---

## 🧮 สูตรคำนวณ

### 👤 แท็บลูกค้า

**ซื้อสินค้าได้สูงสุด**
```
ราคาสินค้าสูงสุด = ยอดเงินในแอป ÷ 0.4
```
> เช่น เหลือ 133 ฿ → ซื้อได้สูงสุด 332.50 ฿

**ยอดที่ต้องจ่ายเอง**
```
จ่ายเอง = ราคาสินค้า × 0.4
รัฐช่วยจ่าย = ราคาสินค้า × 0.6
```
> เช่น สินค้า 100 ฿ → จ่ายเอง 40 ฿, รัฐจ่าย 60 ฿

---

### 🏪 แท็บร้านค้า

**เงินสดที่ต้องทอน**
```
เงินทอน = ยอดสแกน − ราคาสินค้า − (ยอดสแกน × เปอร์เซ็นต์หัก ÷ 100)
```
> หากผลลัพธ์ติดลบ แสดงเป็น 0 ฿ (ไม่ทอนเงิน)
